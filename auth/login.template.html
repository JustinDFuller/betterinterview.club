Better Interviews

The Problem

I've spent years interviewing Software Engineers of all different levels. But, even though I don't lack experience, it never seemed to get easier. I struggled to compare candidates to each other. I never felt that I had a clear "signal" that a candidate was right for the role.

But, then I discovered an incredibly simple way to simplify the decision making process and reduce bias.

How It Works

First, a hiring manager opens a role. They create five "yes" or "no" questions that interviewers will use to evaluate every candidate for the position.

For each candidate, the interviewer receives a feedback request. After the interview, they select "yes" or "no" for the questions.

There is an additional question, asking if the interviewer recommends the candidate.

The hiring manager will receive a summary of each submitted feedback.

Why you should care
